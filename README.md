# Вступление
Данный проект представляет собой фронтенд сайт для [GitHub Pages](https://pages.github.com/), который отображает актуальную информацию о мире, истории сеттинга Дистортум на основе D&D 5 2024.
## Для чего это?
Этот проект в первую очередь показывает мой путь обучения в веб-разработке и в обращении с гитом.
## Что планируется?
Серверная часть.
Система регистрации и авторизации через OAuth2.
Мини-форум.
Трекер инициативы.
Интеграция персонажей с LSS.
Интеграция монстров с dnd.su.
### Средства
HTML 5.
CSS.
JavaScript.
JQuery.
## Что уже реализовано?
Две темы.
Различный вид для разных экранов.
Удобная навигация.
